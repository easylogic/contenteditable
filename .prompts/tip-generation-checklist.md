## 📋 Tip 생성 시 확인사항

### 파일 생성 전
- [ ] 기존 tip 번호 확인 (다음 번호 사용)
- [ ] 관련 시나리오 파일 읽기
- [ ] 관련 케이스 파일들 읽기
- [ ] 기존 유사 tip 확인 (중복 방지)

### 파일 생성 시
- [ ] 영어 버전 파일 생성 (`tip-XXX-{slug}.md`)
- [ ] 한국어 버전 파일 생성 (`tip-XXX-{slug}-ko.md`)
- [ ] Frontmatter 모든 필드 작성
- [ ] ID 형식 확인 (영어는 `-ko` 없음, 한국어는 `-ko` 있음)

### 내용 작성 시
- [ ] Problem 섹션: 문제 명확히 설명
- [ ] Solution 섹션: 최소 2개 방법 제공
- [ ] 코드 예제: 실제 동작하는 완전한 코드
- [ ] Notes 섹션: 주의사항 포함
- [ ] Related Resources: 링크 형식 확인
- [ ] 헤딩에 `{#id}` 패턴 없음

### 검증
- [ ] 린터 오류 없음
- [ ] 브라우저에서 페이지 정상 표시
- [ ] 링크 정상 작동
- [ ] 사이드바에 표시됨
